## Architecture Notes

**Data flow:**
```
Google Sheets (source of truth)
  ↓ daily CSV export from Amanda's PC (renewals data)
  ↓ service account reads (dashboard)
Netlify Functions (serverless API)
  → /api/get-units          reads Sheet1 + Sheet2, returns 131 unit objects
  → /api/get-property-info  reads property info sheet by address
  → /api/update-property-info  writes single field + history row
React (Vite SPA)
  → App.jsx fetches /api/get-units on load, caches with 5min TTL
  → PropertyInfoTab fetches /api/get-property-info per-property on open
```

**Field source split:**
- `source: 'sheet'` fields (washer, dryer, gas, AC type, etc.) come from Sheet2 of the main spreadsheet, parsed at load time, read-only in the UI (🔒 lock icon)
- `source: 'gsheet'` fields (door codes, filter sizes, paint, etc.) come from the property info sheet, editable inline (✏️ pencil icon)
